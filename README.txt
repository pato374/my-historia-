# Mi Historia — Amaro 🌿

## Archivos del sitio web

- `index.html` — La página principal con todo el contenido (historia, imágenes, metas)
- `netlify.toml` — Configuración para Netlify (se usa automáticamente)

---

## Cómo subir a internet GRATIS (Netlify Drop)

1. Entra a 👉 **https://netlify.com/drop**
2. Selecciona la **carpeta "sitio-amaro"** completa y arrástrala a la página
3. ¡Listo! En segundos tendrás un link como `https://nombre-random.netlify.app`
4. Comparte ese link con quien quieras 🎉

---

## Notas
- El sitio funciona en celular y computador
- No necesita internet para abrir (solo para las fuentes decorativas)
- Todo el contenido (historia, imágenes, metas) está dentro del archivo HTML
